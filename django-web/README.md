### python 3.7.9 ( 3.7.10  window 제공 x) 


#### 사용방법 ( 임시 로컬 서버 연결 )
  * 가상환경 설치 (cmd-cd .../django-web 이동)
  ```
  python -m venv "가상환경이름"
  ```
  * 설치한 가상환경 사용
  ```
  myvenv\Scripts\activate
  ```
  * runserver ( 127.0.0.1:8000 사용 )
  ```
  python manage.py runserver
  ```
